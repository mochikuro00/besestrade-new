# BESES TRADE LIMITED - Corporate Website

## Project Overview
- **Name**: BESES TRADE LIMITED (百易貿易有限公司) Official Website
- **Goal**: Establish a trustworthy online presence for an international trading company.
- **Key Features**: 
  - Multi-language support (English, Chinese, Japanese, French)
  - Compliance & Governance section for banking trust
  - High-end "Modern Oriental" corporate aesthetic
  - Secure contact form (No public email/phone)

## URLs
- **Development**: http://localhost:3000

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Backend**: Hono (Cloudflare Workers)
- **Design**: Imperial Red & Gold palette, Glassmorphism, Responsive layouts

## Navigation
- **Home**: Brand overview, video hero.
- **About Us**: Company background and location (Harbour City).
- **Services**: Core trading competencies.
- **Compliance**: AML/KYC info (Crucial for bank accounts).
- **Contact**: Secure inquiry form.

## Deployment
Deployed to Cloudflare Pages.
